In Java Script the Storage Patterns 

1.Variables Temporary storage
It will store only a single value 
2.Arrays Multiple values
It will store multiple values
3.Objects Key-Value Storage
It will store in key-value format
4.Local Storage - Permanent Browser Storage
It will store permanently when the browser refresh or closed the data will be present in the local storage
5.Session Storage - Temporary Session Storage
It will store the data until the browser tab closes
