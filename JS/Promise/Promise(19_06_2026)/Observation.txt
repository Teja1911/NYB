What is DOM Traversing?
DOM Traversing is the process of navigating between related elements in the DOM tree such as parent, child, and sibling elements. It is useful when we don't have a direct reference to an element but can access it through a related element. In real-world applications, DOM Traversing is commonly used in dynamic components such as product cards, tables, and forms where user actions occur on child elements but information must be retrieved from parent or sibling elements.

What is Dynamic DOM Creation?
Dynamic DOM Creation is the process of creating HTML elements at runtime using JavaScript instead of hardcoding them in HTML. This is commonly used when data comes from APIs, databases, or user input. Methods such as createElement(), appendChild(), and append() allow us to build and insert elements dynamically into the DOM.

What are Event Listeners?
Event Listeners are mechanisms that allow JavaScript to respond to browser or user-generated events such as clicks, input changes, form submissions, and keyboard interactions. Asynchronous JavaScript
What Problem Does It Solve?
JavaScript is single-threaded, meaning it can execute only one task at a time.
Until the loop completes:
Browser freezes
User cannot click
UI becomes unresponsive
This creates a poor user experience.
Why Is It Needed?
Many operations take time:
API Calls
Database Requests
File Uploads
Image Loading
Timers
We don't want the browser to stop working while waiting.
Instead:
Start Task
Continue Other Work
Execute Result Later
This is asynchronous programming.
Real-Time Usage
Google Search
Payment Processing
File Upload
Fetching Products
Loading User Data
Advantages 
Non Blocking Execution
Better user experience
Faster applications
Limitations
More complex than synchronous code
can create callback hell

A design pattern where a single parent listener manages events for multiple child elements using event bubbling.
Callback Function
A function passed as an argument and executed after another operation completes.
Throttling
Limits execution frequency to improve performance during high-frequency events.
