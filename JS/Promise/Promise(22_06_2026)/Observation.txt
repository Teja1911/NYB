Comparison Table
| Method               | Waits For     | Failure Behavior       |
| -------------------- | ------------- | ---------------------- |
| Promise.all()        | All Success   | One Fail = Fail        |
| Promise.race()       | First Settled | Success or Fail        |
| Promise.allSettled() | All Results   | Never Fails            |
| Promise.any()        | First Success | Fails Only If All Fail |
