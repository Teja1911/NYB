Why Web Application are needed?
1.Cross platform support (i.e laptop desktop tablet mobile)
2.No Installation required (i.e Web apps open instantly using URL)
3.Easy maintenance and updates(i.e Server update once all user automatically get the latest version) 
4.Lower Development Cost (i.e Single codespace works everywhere)
5.SEO and Searchability (i.e Google search or browser search)
6.Instant Accessibility (i.e Anyone with internet + browser can use it)
7.Faster Deployment (i.e Netlify Vercel Github)
8.Easier for beginners and companies
