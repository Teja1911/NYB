git:
Git is a Distributed Version Control System.
It tracks changes made to your project over time, allowing you to save versions, collaborate with others, and restore previous versions if needed.

You accidentally delete everything.
Without Git:
Project Lost ❌
With Git:
Restore Previous Version ✅
This is the main problem Git solves.

Without Git,
everyone edits the same files and overwrites each other's work.
With Git,
every developer works independently, and Git combines everyone's changes.

Version Control
Version control means keeping the history of every change made to a project.

Command	Purpose
git init	                    Initialize a repository
git status	                    Show current status
git add .	                    Stage all files
git add fileName	            Stage one file
git commit -m "message"	        Create a commit
git log	                        View commit history
git remote add origin URL	    Connect to GitHub
git push -u origin main	        First push
git push	                    Push later changes
git pull	                    Download latest changes
git clone URL	                Copy a repository locally
